# WhatsApp Controller

A WhatsApp automation and management dashboard built with React, TypeScript, and Tailwind CSS.

## Project Structure

- `src/` - Frontend React application
- `backend/` - Node.js backend for WhatsApp integration

## Technologies

- Vite
- TypeScript
- React
- Tailwind CSS

## Getting Started

```sh
# Install dependencies
npm i

# Start development server
npm run dev
```

## Backend Setup

```sh
cd backend
npm i
npm start
```
